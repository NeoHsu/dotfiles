## Installation
> **Warning: the dotfiles only for macOS** 

### Using Git

```shell
git clone --recursive https://github.com/NeoHsu/dotfiles.git ~/.dotfiles
```

### Using dotbot to set dotfile's links

```shell
cd ~/.dotfiles
chmod -x install
./install
```
